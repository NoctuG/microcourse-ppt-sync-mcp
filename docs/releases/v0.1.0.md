# v0.1.0 Release Notes

v0.1.0 是 Microcourse PPT Sync MCP 的第一个 MVP Release，范围只包含 PPT 背景视频闭环。

## 支持能力

1. **读取 Timeline**：读取用户提供的 `timeline.json`，解析幻灯片、动画触发时间、动画持续时间和自动翻页时间。
2. **写入 PPT 动画计时**：将 Timeline 写入已有 PowerPoint 动画序列，使幻灯片放映可按时间轴自动播放。
3. **导出 PPT 背景视频**：调用 PowerPoint 桌面版原生视频导出能力，生成保留 PPT 动画效果的背景视频。

## 运行环境

- Windows 10/11
- Microsoft PowerPoint 桌面版
- Python 3.11+

## 不在本 Release 范围内

v0.1.0 不包含 ASR、字幕格式解析、剪映自动化、人像处理或最终视频合成。
